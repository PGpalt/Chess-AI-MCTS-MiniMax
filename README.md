Monte Carlo Tree Search and MiniMax Algorithms were implemented as AI agents to a variation of the game of chess. Artificial Inteligence class of Technical University of Crete.
To execute run the udpServer once and 2 clients. Black is using MCTS an algorithm that can be implemented with zero knowledge of the game by using simulations while White is using
the MiniMax algorithm with an evaluation function which evaluates the position at each step and thus Game Knowledge is required.
